Feste

Email Subscription Management

It’s important for applications to give users the option of unsubscribing from emails. I started Feste as an open source project that gives users that freedom, while still providing options for them to stay informed. Feste is a gem that gives users the option to unsubscribe from emails. But unlike other alternatives, which only allow for an all-or-nothing approach, Feste lets administrators sort individual emails into broader categories. The user is given a dashboard where they can manage which categories they would like to keep receiving, and which they would like to unsubscribe from.
